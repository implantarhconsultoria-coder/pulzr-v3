# PULZR Mobile Flutter

App mobile Flutter do PULZR com identidade preto ultra-escuro + verde neon.

## O que esta implementado

- Feed social no estilo Instagram + Strava.
- Cards de atividade com atleta, foto do treino, metricas em destaque, legenda e comentario.
- Perfil do atleta com avatar, nivel, conquistas e historico de treinos.
- Tela de atividade em tempo real com cronometro, distancia, ritmo, calorias, BPM e botoes FINALIZAR/PAUSAR.
- Bottom navigation fixa com Feed, Desafios, logo PULZR, Treinos e Perfil.

## Como rodar

```bash
cd mobile_flutter
flutter pub get
flutter run
```

## Observacao

O ambiente do agente usado para criar esta entrega nao tinha Flutter/Dart instalado. Por isso o codigo foi enviado ao GitHub, mas a compilacao precisa ser feita em uma maquina com Flutter instalado ou em CI configurado com Flutter.
